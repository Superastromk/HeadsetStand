# HeadsetStand
A 3D-Printable headset stand in the theme of an eigth note, with my choir program listed on it.
# Why I Made This
Since I use it most to listen to music, I decided to keep a music theme for this. I designed it like an 8th note, and since it reminded me of the choir program I love to be a part of, I ended up putting that in too.
